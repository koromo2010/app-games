# ノーザンブランチ 個人利用プロトタイプ

公開情報をもとに、ゲームの大枠と画面操作を検証するために作った個人利用向け試作です。市販品の完全再現ではありません。

## 実装済み

- 2～4人で1台の端末を順番に操作
- 手札上限7枚と手番プレイヤーだけの手札表示
- 資源獲得、生産、価値による売買
- 建物の勝利点と各ターン1回のブラウニーアクション
- 豚・鶏によるダング発生
- 10勝利点で即勝利
- ブラウザー内への途中保存

## 仮データと未実装

商品レシピ、カード枚数、建物名、建設費、勝利点、建物効果は仮設定です。正式データ入手後は `lib/northern-branch-data.ts` を中心に差し替えます。ネットワーク対戦、正式なカード90枚、人数別在庫、特殊カード、商会コンテスト拡張は未実装です。

## 個人利用枠の解除

Vercelにサーバー側環境変数 `PRIVATE_GAME_ACCESS_KEY` を設定します。トップページ上部の無注釈フィールドへ同じ文字列を入力すると、サーバー照合後に個人利用ゲームが表示されます。解除状態は30日間のHttpOnly Cookieに保存され、未解除で `/northern-branch` を直接開くとゲームロビーへ戻ります。
