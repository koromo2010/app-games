# 暗号傍受（仮）暫定仕様

## 公開状態

- 技術ID: `code-intercept`
- URL: `/code-intercept`
- `private: true` の検証用オンラインゲーム
- 表示名・用語・演出は公開前に変更する

## 初期ルール

- 4人以上、赤・青の2チーム。各チーム2人以上、人数差は1人以内。
- 各チームは番号1〜4の秘密単語を持つ。
- 毎ラウンド、チーム内で順番に交代する出題者へ、重複なしの3桁暗号を表示する。
- 出題者は暗号順に3つのヒントを提出し、両チーム分が揃ってから同時公開する。
- 出題者以外がチームとして味方暗号を1件回答する。第2ラウンドからは敵暗号も1件回答する。
- 味方回答の不正解・未回答は自チームへ1ダメージ。敵の傍受成功は2ダメージ。
- 両チームのダメージを同時反映する。片方だけ0点なら相手の勝利、両方0点なら引き分け。
- 初期ポイントは5点。秘密カードはゲーム終了まで固定する。
- 時間制限は初期検証では設けない。未回答を自動失敗にする秒数はプレイテスト後に決める。

## 秘密情報

- 秘密単語は同じチームにだけ返す。ゲーム終了時のみ全チーム分を公開する。
- 現在の暗号は出題者にだけ返し、ラウンド結果で公開する。
- ヒントは両チームが提出するまで敵へ返さない。
- 味方回答と傍受回答はラウンド確定まで相手チームへ返さない。
- 得点とダメージはサーバーで正解暗号から再計算する。
