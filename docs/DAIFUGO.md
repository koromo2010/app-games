# 大富豪（オンライン対戦＋CPU練習）

## 現在の遊び方

- 通常はログインした3〜6人で、4文字の部屋コードと任意の合言葉を使ってオンライン対戦する。
- `/daifugo/practice` では、1人＋CPU3人の練習対戦ができる。

- 1人のプレイヤーとCPU3人で、ジョーカー1枚を含む53枚をすべて配る。
- ダイヤの3を持つプレイヤーから開始し、最初の組にはダイヤの3を含める。
- 強さは `3 < 4 < ... < K < A < 2 < Joker`。
- 場が空なら、1枚または同じ数字の組を最大4枚まで出せる。4枚＋ジョーカーの5枚組は使わない。
- 場に組があるときは、同じ枚数でより強い組だけを出せる。
- ジョーカーは単独なら最強。同じ数字の組では、その数字のワイルドカードとして扱う。
- パスしたプレイヤーは、その場が流れるまで再参加しない。ほかの全員がパスすると、最後にカードを出したプレイヤーから新しい場を始める。そのプレイヤーが上がっていれば次の未上がりプレイヤーから始める。
- 手札をなくした順に、大富豪、富豪、貧民、大貧民とする。

## 実装境界

- `lib/playing-cards.ts`: デッキ生成、シャッフル、配札、手札からの安全な取り出し。
- `lib/daifugo.ts`: 強さ、組、合法手、パス、場流し、順位、CPU選択の純粋ロジック。
- `lib/daifugo-room-domain.ts`: サーバー正本の開始、合法手、パス、時間切れ処理。
- `lib/daifugo-room-store.ts`: Redis保存、revision競合防止、1人1部屋、TTL、再接続。
- `lib/daifugo-room-normalizer.ts`: 保存データの検証と復元。
- `app/daifugo/DaifugoGame.tsx`: オンラインの入室、ロビー、対戦、結果画面。
- `app/daifugo/DaifugoPractice.tsx`: ブラウザ内のCPU練習。
- `app/daifugo/DaifugoTable.tsx`: 場、相手の手札枚数、手番の表示。
- `app/daifugo/DaifugoRulesDialog.tsx`: 画面内ルール。

オンライン対戦ではシャッフル、全手札、合法手判定をサーバー正本とし、APIは閲覧者以外のカードIDを返さない。結果、レーティング、プレイ時間、本人用プレイバックを冪等に記録する。CPU練習はアカウント戦績へ含めず、完了時間だけを匿名の実プレイ時間統計へ送る。
